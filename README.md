# Depth Classifier Benchmark

This repository contains the results and scripts for the paper:

"A Comparative Benchmark of Depth-Based and Classical Classifiers: Performance, Robustness, and Representation Effects"

## Structure

- `scripts/`: Benchmark and visualization scripts
- `analysis/figures/`: Figures used in the paper
- `analysis/tables/`: Tables and summaries
- `.csv`: Benchmark outputs
- `results_full.rds`: Full results object
- `session_info.txt`: R session for reproducibility

## Reproducibility

The repository preserves the original folder structure used during the experiments to ensure exact reproducibility.
